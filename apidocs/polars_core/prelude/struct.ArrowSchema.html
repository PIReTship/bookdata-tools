Expand description

An ordered sequence of Fields with associated Metadata.

Schema is an abstration used to read from, and write to, Arrow IPC format, Apache Parquet, and Apache Avro. All these formats have a concept of a schema with fields and metadata.

Fields

fields: Vec<Field, Global>

The fields composing this schema.

Optional metadata.
